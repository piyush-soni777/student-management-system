# 🎓 Student Management System

Student Management System is a Java-based application designed to manage student records efficiently.  
It allows users to perform operations like adding, viewing, searching, updating, and deleting student data.

## Run in 2 steps

```cmd
javac StudentManagement.java
java StudentManagement
```

That's it!

## Features
- Add / View / Search / Update / Delete students.
- Search by name or course.
- Color-coded GPA table.
- Statistics with course bar chart
- 5 sample students pre-loaded.

## Requirements
- Java 14 or higher (uses switch expressions)
