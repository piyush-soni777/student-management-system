# Student Management System (Single File)

A simple CLI-based Student Management System in a **single Java file** — no packages, no Maven needed.

## Run in 2 steps

```cmd
javac StudentManagement.java
java StudentManagement
```

That's it!

## Features
- Add / View / Search / Update / Delete students
- Search by name or course
- Color-coded GPA table
- Statistics with course bar chart
- 5 sample students pre-loaded

## Requirements
- Java 14 or higher (uses switch expressions)
